Class: SmartChat

SmartChat(env, conversation_id, data)

A chat session in a SmartChat environment.

Base class format follows ChatML JSON spec. Subclasses should override to_chatml, from_chatml, and parse_user_message to provide file-type specific parsing and formatting.
